# Architecture Review Board

## Purpose

The architecture review board reviews significant cloud and multi-cloud decisions for alignment to standards, risk, and operational supportability.

## Board Responsibilities

- review new or changed cloud designs
- assess exceptions
- confirm control coverage
- resolve design risk

## Board Outputs

- approved
- approved with conditions
- rejected
- more information required

## Board Rule

The board should make architecture decisions consistent across cloud providers, not re-invent the review standard each time.
